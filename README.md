# My_Portifolio..
Lets delve into my portifolio [website]
A modern personal portfolio website built with **React** and **Vite**. Showcases projects, skills, problem-solving track record, and ways to connect.

## Live Demo
- **Website URL:** [Click here](https://ganapathi-jahnavi-durga.netlify.app/)


## Features

- **Hero** — Intro section with quick overview
- **About** — Background and expertise summary  
- **Skills** — Technical competencies displayed
- **Projects** — Showcase of work and GitHub links
- **Problem Solving** — LeetCode profile & project highlights
- **Contact** — Reach out options with dark pink theme
- **Theme Toggle** — Light/dark mode switch
- **Responsive Design** — Works seamlessly on all devices
