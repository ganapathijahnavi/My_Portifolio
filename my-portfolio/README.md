# Illustration Portfolio

Single-page portfolio built with Vite + React and a bold illustration-style layout.

## Run locally

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`
4. Preview build: `npm run preview`

## Update content

- Projects: [src/data/projects.js](src/data/projects.js)
- Skills: [src/data/skills.js](src/data/skills.js)
- Hero/About/Contact text: [src/sections](src/sections)

## Notes

- Illustration assets are placeholders and can be replaced in [src/assets](src/assets).
